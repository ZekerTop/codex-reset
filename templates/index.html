Codex 额度控制台

粘贴 ChatGPT session JSON，查询邀请资格、reset credits 和当前使用状态。

Session JSON

内容只发送给本地 Flask 服务，由后端转发到 ChatGPT API。

不存储你的 Session JSON。 仅用于当前页面发起的查询和重置请求，不会写入本地文件或数据库。

快捷键：⌘ / Ctrl + Enter

获取 Session

按下面三步复制响应内容。

  1. 1 浏览器登录 chatgpt.com
  2. 2 打开 DevTools 的 Network，访问 https://chatgpt.com/api/auth/session
  3. 3 复制 Response JSON，粘贴到左侧输入框后点击查询。
提示：如果查询失败，先确认代理可访问 ChatGPT，再刷新页面重试。
